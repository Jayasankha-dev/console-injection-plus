# Console Injection+ 🚀

A powerful and lightweight Chrome extension designed for developers and power users to inject custom JavaScript or CSS snippets directly into any webpage via the browser console environment.

[![Chrome Web Store](https://img.shields.io/badge/Chrome_Web_Store-Available-green?style=for-the-badge&logo=google-chrome)](https://chromewebstore.google.com/detail/console-injection+/odcpgbablphlgghjlfinipbpcgijdahk)

## 🌟 Overview

**Console Injection+** simplifies the process of testing, debugging, and enhancing web pages on the fly. Whether you are a web developer testing UI changes or an automation enthusiast, this tool provides a seamless interface to run your scripts without manually opening the DevTools every single time.

## ✨ Features

* **Instant Injection:** Inject JS/CSS with a single click.
* **Persistent Scripts:** Save your favorite snippets for future use.
* **Clean UI:** Simple and intuitive interface that doesn't get in your way.
* **Developer Friendly:** Built by a developer, for developers.

## 🛠️ Installation

You can install the extension directly from the Chrome Web Store:

👉 **[Download Console Injection+ Here](https://chromewebstore.google.com/detail/console-injection+/odcpgbablphlgghjlfinipbpcgijdahk)**

### Manual Installation (For Development)
1.  Clone this repository: `git clone https://github.com/Jayasankha-dev/YOUR_REPO_NAME.git`
2.  Open Chrome and navigate to `chrome://extensions/`.
3.  Enable **Developer mode** (top right).
4.  Click **Load unpacked** and select the project folder.

## 🚀 How to Use

1.  Open the extension popup from your Chrome toolbar.
2.  Paste your JavaScript or CSS code into the editor.
3.  Click **Inject** to see the changes live on your current tab.
4.  (Optional) Save snippets to use them later on different sites.
